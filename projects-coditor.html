Coditor

DATE:

April 2020

DESCRIPTION:

Coditor is a note-taking tool web application for computer science students. The idea stemmed from the struggles of trying to take notes on GoogleDocs and having auto-correct destroy the formatting of the code chunks. When typing on a Coditor Document, in addition to the option to bold, italicize, underline texts, users can apply "Coditor" style to text. Coditor text correctly recognizes code chunks and formats into pretty printed code with appropriate indentations and color coded classes, variables, etc.

This web application allows users to Sign-up with an account and Login through authenticated and valid login information that is verified through stored user info in the MySQL database. Once logged in, the home page opens to the user's "Drive" which stores all their documents (MySQL) in two tabs, one for their documents that they are the owner of and the second for the shared documents. Each document could be shared with another user via email. If the email is pre-registered than that shared email has access to also edit the document. If the email is not recognized, the shared email has guest view only access to the document.

Technologies:

HTML/CSS, JavaScript, Java, JSP, MySQL, Eclipse, Tomcat
