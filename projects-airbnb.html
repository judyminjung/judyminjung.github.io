Mock Airbnb App

DATE:

July 2019

DESCRIPTION:

For this mock Airbnb web application project I created two versions: a consumer app for guests exploring bookings and a provider app for hosts putting up their listings.

The frontend is built using Ionic and is connected to a mySQL database using Express.js and Angular. The functionality included login, registration, profile page, explore page, and details page. Each user is stored in the admin database for login authentication or to edit during registration which is later used to load the user's profile page. The property listings are stored in a separate database which is used to load the explore and furthermore a specifically selected listing's details page.

TECHNOLOGIES:

HTML, CSS, JavaScript, TypeScript, Ionic 4, Angular 8, MySQL, Loopback
