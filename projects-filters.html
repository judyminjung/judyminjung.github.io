Image Filters

DATE:

March 2019

DESCRIPTION:

This is a course project from University of Southern California's CSCI 103 course (Spring 2019). This program performs simple kernel based image processing filters on an image. The three filters are 1.) the Sobel operator for edge detection (using a fixed sized kernel), 2.) the Gaussian blur filter for low-pass filtering, and 3.) unsharp mask filter for sharpening an image (which uses the Gaussian blur filter).

Program:

C++ 11

Concepts:

Image processing, Dyanmically allocated 2D arrays
